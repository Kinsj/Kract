# 介绍
这是一个手动实现的简易版react
其中包含了以下特性：
* createElement
* render
* 并发模式
* Fiber
* 集中渲染
* Reconciliation
* Function Component
* Hooks

# 运行方式
```shell script
git clone git@github.com:Kinsj/Kract.git
cd Kract
yarn
yarn dev
```
